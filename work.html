Research

Galaxy Classification Using Machine Learning

Applied Computer Vision, Transfer Learning, and Deep Metric Learning to advance automated galaxy classification using large-scale astronomical image data. Pretrained models on the massive citizen-science Galaxy Zoo dataset, then fine-tuned on the expert-labelled Revised Shapley-Ames catalogue to improve morphological precision. Bridged crowd-sourced learning and expert taxonomy to classify galaxies according to the Hubble tuning fork system with enhanced representation learning.


Machine Learning Competitions

West Africa Land Cover Classification

Top 10 solution in a Zindi Earth Observation challenge classifying forest, cocoa, and palm plantations using 3-year Sentinel-2 satellite data. Built a full deep learning pipeline (CNN & CNN-LSTM) for multi-temporal land cover classification, optimised using weighted F1 score. Applied remote sensing and temporal modeling to support forest conservation and deforestation monitoring in Africa.
